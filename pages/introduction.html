Introduction

Node.JS is a powerful cross-platform runtime environment that is designed for developing server-side web applications. In this tutorial we are going to provide a basic introduction into how Node.JS works and how you can start developing your own applications today.


What is Node.JS?

Node.JS, like we mentioned above, is a cross-platform runtime environment. It is designed for developing server-side web applications. Node.JS applications are written in javascript and can be run on any platform. The javascript is executed by the V8, a javascript engine developed by Google. This is the same engine that Chrome uses inside of its browser.

Why use Node.JS

Node has a very powerful attribute that sets it apart from other web frameworks that gives it a huge advantage. Instead of just creating the front end html and relying on stateless scripts on the server, you acctually create your own web server that allows you to rely on a persistant state. So what does this mean? Well most web developers are probably used to using apache. You configure apache to listen on port 80 and then when your clients go to your web page. Apache finds the file they want and then serves them the file.

Where does it come from?

Bootstrap was developed at Twitter by Mark Otto and Jacob Thornton. They released it as an open source project August 2011. Once it became open sourced it became so popular that it was the #1 project on Github in 2014.

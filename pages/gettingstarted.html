Getting started

Like we mentioned on the previous page node.js is a server side application. Applications are written in javascript and so the first thing we need to get is the javascript parser.

Ubuntu

To install node on ubuntu you need to issue the command sudo apt-get instal nodjs

Mac OS X

To install node on Mac you need hombrew and then you can issue the command brew install node

Once the interpreter is installed you can invoke it using node <filename.js>

Introduction Program

The first thing were going to do is create a basic javascript program to get us started. Thets create a file called helloworld.js

            
/*Prints hello world to the screen*/
console.log('Hello World');

Next were going to run this by typing node helloworld.js You should see the command line print "Hello World" and then exit.

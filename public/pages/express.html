In order to maximize the potential of Node JS, you will want to consider the many modules that can be used with it. One of the most popular modules is Express JS. In fact, express is part of a very common JS based software stack, called MEAN(Mongo,Express,Angular,Node).

In order to install a module, you will need to use the "npm" command. This should be available with your basic Node install. To install express, use the following command: 


        npm install express
You should now see that you have a "node_moduels" directory, containing, at the least, an express directory. With express we can very easily spin up a simple static file server. Consider the code below: 


        var express = require('express'),
        app = express();

        app.use(express.static(__dirname + '/public'));

        app.listen(8080);

Running this code creates a static server that serves up the public directory. Placing a simple index.html in this location will allow you to create a basic website, hosted via node JS.
